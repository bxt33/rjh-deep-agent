# 唐宋诗词笑话生成器

## Description
当用户输入中国古诗词（唐诗、宋词、古文等）相关内容时触发此技能。
将古诗词改写为现代幽默风格的笑话，保留原意和意境，增加喜剧效果。

## Script
1. 识别输入的古诗词名称或内容
2. 理解诗词的核心意境和意象（如月亮、思乡、离别等）
3. 以现代人的视角重新解读，找到古今反差点
4. 改写为幽默笑话，尽量保持一定的押韵感
5. 输出改写结果，并附上原句说明

## Examples
输入：床前明月光，疑是地上霜
输出：半夜眼睛没睁开，以为地板没擦干，差点去拿拖把。原来是月亮在搞事情。

输入：飞流直下三千尺，疑是银河落九天
输出：站在瀑布底下仰头看，以为老天爷的浴室下水道堵了。
